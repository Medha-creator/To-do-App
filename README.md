# To-do-App
The To-do App is a simple task management application built using HTML, CSS and JavaScript. It allows users to add tasks along with due dates, display them dynamically on the page, and delete completed tasts.
This project mainly focuses on JavaScript concepts.

**Features**
- Add new tasks easily.
- Set due dates for better task planning.
- Delete tasks when they are no longer needed.
- Simple and user-friendly interface.

**Technologies Used**
- HTML
- CSS
- JavaScript

**JavaScript Concepts Implemented**
- Arrays and Array Methods
- Variables
- Objects
- Functions
- DOM Manipulation
- Loops
- Object Destructuring
- Event Handling
- Dynamic HTML Rendering

**How to Run the Project**
- Download or clone the repository.
- Open the project folder in VS Code.
- Open the HTML file in a browser using Live Server.
- Enter a task and select a due date.
- Click the "Add" button to add the task.
- Use the "Delete" button to remove a task.
